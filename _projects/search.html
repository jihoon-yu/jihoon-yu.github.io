자주 사용되었던 Mac OS, iPad OS의 시스템 검색 기능인 Spotlight 에서 간단한 탐색 기능만을 제공하는 지점이 아쉬움으로 느껴졌습니다. OS에서 직접 제공하는 검색 기능을 더 강력하고 직관적인 방향으로 지원하게 된다면 어떤 형태일지, 지금 보다 더 편리하게 사용할 수 있을지 호기심에서 개선의 컨셉 디자인을 제안하게 되었습니다.
Simple Flow & Fast Approach
기존 Spotlight 의 단순한 검색 플로우를 최대한 유지하고자 하였습니다. 검색 결과에서 간단한 정보들을 제공하여 원하는 정보를 빠르게 탐색할 수 있도록 의도하였습니다. 또한 원하는 검색결과를 더 자세히 보고 싶을 경우 바로 외부 어플리케이션으로 이동하는 이탈 플로우에서 검색 결과 모듈을 제공해서 사용자들의 목표를 더 빠르게 획득 할 수 있도록 설계했습니다.
Drag and Drop Flow
검색을 사용하게 되면 Background Blur를 활용, 검색 결과에 집중 할 수있는 환경을 제공합니다. 사용자들은 검색 모듈에서 검색 결과의 간략한 정보를 확인할 수 있습니다. 검색 결과 리스트를 모듈 외부로 드래그&드랍을 하게 된다면, 자연스럽게 원하는 정보를 더 자세하게 확인해 볼 수 있습니다.
Various Search Modules
기존의 간략한 프리뷰만 제공하고 외부 어플리케이션에서 추가 탐색의 과정을 거쳤던 불편함을 개선하고자, 자세한 정보를 일관되게 보여줄 수 있는 모듈의 형태로 디자인을 진행하였습니다. 각 모듈들은 앞서 이야기 되었던 통일된 인터페이스, 디자인 규칙을 바탕으로 제공됩니다. 이를 바탕으로 사용자가 어떤 검색 결과를 확인해도, 시각적으로 통일된 시스템 안에서 머무르고 있다는 경험을 제공하고자 하였습니다.
